## init_demo_v1
### 基本说明
完成了前端显示的构建和基本功能实现：<br/>
包括以下三个部分：
1. 项目介绍
2. 模型查看：根据模型label查看三维模型
3. 草图检索：根据草图显示三维检索结果

其中，三维显示所用模型为本地文件，仅用于测试显示及交互状态。
